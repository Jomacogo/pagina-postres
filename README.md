# 🍰 Página de Postres

Landing page desarrollada con React + Vite para la presentación de un emprendimiento de venta de postres.

## 🚀 Demo
👉 [(https://elrincondelospostres.netlify.app](https://elrincondelospostres.netlify.app)

---

## 🧁 Descripción

Este proyecto consiste en una página web tipo landing donde se muestran diferentes postres disponibles, información del emprendimiento y formas de contacto.

Fue desarrollado con el objetivo de practicar y mejorar habilidades en desarrollo frontend, diseño responsive y organización de componentes en React.

---

## 🛠️ Tecnologías utilizadas

- React
- Vite
- JavaScript
- CSS (Flexbox y Grid)

---

## 📌 Características

- Diseño responsive 📱
- Navbar fija (sticky)
- Sección de productos organizada en grid
- Integración con WhatsApp para contacto
- Estructura de proyecto escalable
